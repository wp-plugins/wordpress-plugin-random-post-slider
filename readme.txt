Random post slider wordpress plugin create a post slider on the wordpress website. We have option to customize the slider direction from Scroll Right, Scroll Left, Scroll Up, Scroll Down. This slider use jquery cycle script for the slider effect. the live demo available on my home page.

= Features of this plugin =

*   Easy to customize.
*   Support all browser.
*   Short code available for pages.
*   Hyperlink option to each post.
*   Separate css file to change the style.

We have three option to install and configure this Image horizontal reel scroll slideshow plugin.
